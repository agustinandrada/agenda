
# AGENDA DE CONTACTOS

Sistema que permite el alta, baja y modificacion de contactos.


## Stack Tecnológico

Se utilizó Laravel 12 con PHP, el proyecto es en monolito por lo tanto backend como frontend utilizan esta tecnología.

Para las vistas se utilizó blade con Boostrap 5 y Boostrap-Icons.

Para la Base de Datos se utilizó MySQL



## Instalación


```bash
  cd agenda
  npm install
  composer install
```
    
## Migración de Base de datos



```bash
  php artisan migrate
```


## Run project

Para levantar el proyecto en local

```bash
  php artisan serve
```
